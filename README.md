# 🍕 Pizza Sales Dashboard (Power BI)

## 📊 Overview
This project presents an interactive Power BI dashboard analyzing pizza sales data for a UK-based franchise. The dashboard provides insights into sales performance, customer preferences, and operational trends.

## 🚀 Key Features
- Sales and revenue tracking
- Orders analysis by time (hour & session)
- Category-wise and size-wise performance
- Weekly sales trends
- Identification of top and least performing pizzas

## 📈 Key Insights
- Total pizzas sold: **49,574**
- Total revenue generated: **$817,860**
- The **Classic category** contributed the highest sales
- **Evening time** recorded the highest number of orders
- **Large-size pizzas** had the highest demand
- **Friday** is the busiest day of the week
- Dashboard highlights **top 3 and bottom 3 pizzas**

## 🛠 Tools Used
- Microsoft Power BI
- Data Analysis
- Data Visualization

## 💡 Business Insights
- Peak evening demand suggests optimizing staffing during those hours
- High demand for large pizzas indicates customer preference for value deals
- Friday spikes can be leveraged for targeted promotions

The dashboard helps stakeholders make data-driven decisions by identifying sales patterns, customer behavior, and product performance.
